Dict 卡片「通俗理解」黄色框 4 种替代方案

当前线上的版本（黄色填充）放在最上面做对比，下面 4 个 variant 名称清楚标注。

Baseline
当前线上版本（黄色填充 + 左边 3px 金条）
.dict-plain { background: color-mix(in srgb, var(--accent-gold) 10%, transparent); border-left: 3px solid var(--accent-gold); }
Agent Loop核心概念

Claude Code 的核心执行循环，由「读取 → 思考 → 行动 → 观察」四步组成。

💡 想象一个厨师在厨房工作：先扫一眼订单（读取），想做什么菜（思考），动手切炒（行动），然后尝一口看咸淡（观察），周而复始。
Tool Use能力机制

LLM 通过结构化输出调用外部工具（如文件读写、终端命令、网络请求）的能力。

💡 就像你请了一位翻译官，但他不会亲自开车——他会写一张纸条给司机（调用工具），司机照做后回来汇报结果。
Context Window模型参数

LLM 单次推理时可处理的最大 token 数量，决定了「同时记住多少信息」的上限。

💡 好比考试时的草稿纸面积。纸越大，你能同时写下和参考的公式、步骤就越多；纸小了就得边写边擦。
Variant 1
极淡黄底 + 无边框
