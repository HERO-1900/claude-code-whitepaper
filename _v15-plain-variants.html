Agent Loop n.

Claude 在单次用户请求内部的「思考→行动→观察→再思考」循环。与外部多轮对话不同，这是模型在后台自主驱动的决策链。

像一名厨师接到"做一道菜"的指令后，不是立刻下锅，而是先在脑中拆解：选食材→切配→火候→调味→装盘，每一步做完再评估下一步。

Prompt Caching n.

Anthropic 提供的 API 特性，允许开发者在多轮请求中复用前文的 prompt 前缀，降低 token 成本与延迟。

就像老师批改同一批学生的作业，第一本该看教学大纲，后面 39 本直接翻答案页——大纲只读一次。

Context Window n.

大模型单次推理时可接收的最大 token 数量。Claude 4 Opus 支持 256K 上下文，相当于约 20 万汉字。

相当于你同时能摊开在桌面的纸张总面积。纸越大，能同时参考的材料越多，但翻找特定信息也越容易眼花。
