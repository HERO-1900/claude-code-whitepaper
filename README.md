# Claude Code 白皮书 · 架构分析

> 一份社区驱动的 Claude Code 架构分析与阅读指南。
> 面向开发者、研究者与 AI Agent 系统设计者。

## 项目定位

本项目是对 **Claude Code（Anthropic 出品的 AI 编程 CLI）**做源码级架构分析的公开读物，不是对 Claude Code 本体的再发布，也不是替代品。所有讨论聚焦在：

- **系统如何被组织**：模块、子系统、数据流、生命周期
- **工程哲学与取舍**：为什么选 A 不选 B，付出了什么代价
- **可迁移的模式**：对其他 Agent 系统、CLI 工具有借鉴价值的做法

项目把工程视角（架构/数据流/性能）与产品视角（UX/能力边界/副作用）结合，力图给读者一份「读完就能自己搭类似系统」的深度参考。

## 如何阅读

线上版：**<https://hero-1900.github.io/claude-code-whitepaper/>**

- 建议从 `part0` 序章开始，先建立整体心智模型
- `part2` 的「代码架构完全解构」与「好奇心驱动的深度问答」是两条并行的主线
- `part3` 各子系统可按需点选
- `part4` 工程哲学与 `part5` 批判与超越是整体思辨
