纵深防御原则 Defense in Depth Principle
Claude Code 采用"无空白单元"策略：4 层防御中每个威胁向量至少有一个 强 防御。单层失效不会导致系统完全沦陷，攻击者必须同时突破多层才能造成实质危害。安全相关代码占总代码库的 32.7%（621/1,902 文件），权限关键词出现 5,160 次。 Claude Code uses a "no empty cells" strategy: each threat vector has at least one Strong defense across 4 layers. Single layer failure does not cause system compromise; attackers must breach multiple layers simultaneously. Security-related code covers 32.7% of the codebase (621/1,902 files); permission keywords appear 5,160 times.
