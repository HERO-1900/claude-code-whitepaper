6 个核心概念6 Core Concepts
Orchestrator
中央编排者 · 主 Claude Agent，拆分任务并分配给 Worker。 Central coordinator — the main Claude agent that decomposes and dispatches tasks.
💡 通俗理解：项目经理派活 Analogy: Project manager assigning tasks
Worker Swarm
4 个 Worker 组成"蜂群"，并行执行：搜索 / 编辑 / Shell / 分析。 4 workers form a swarm running in parallel: Search / Edit / Shell / Analysis.
💡 通俗理解：四个专岗同事同时开工 Analogy: Four specialists working in parallel
Scratchpad
共享白板 · 所有 Agent 读写的中间状态存储区。 Shared memory area all agents read/write intermediate state.
💡 通俗理解：办公室公共白板 Analogy: Office shared whiteboard
Completion Queue
完成队列 · Worker 汇报结果按 FIFO 排队，编排者逐个聚合。 FIFO queue of worker results; orchestrator aggregates them in order.
💡 通俗理解：银行叫号窗口 Analogy: Bank teller's ticketing queue
1 + N Pattern
1 个编排者 + N 个 Worker，N 最大为 10，防止上下文爆炸。 One orchestrator + N workers (max 10) to bound context growth.
💡 通俗理解：领班 + 小队的固定编制 Analogy: Foreman + fixed-size crew
Bidirectional IPC
双向通信 · 派发 (紫) 与回报 (青) 各走一条路径，带动画粒子。 Two-way comm — violet dispatch + cyan report lanes with animated particles.
💡 通俗理解：对讲机双工通话 Analogy: Full-duplex walkie-talkie
