四种 Agent 拓扑 — LEGO 组合哲学 Four Agent Topologies — LEGO Composition Philosophy

串行链 · Pattern 01 Sequential Chain · Pattern 01
串行最简单 Simplest to Reason About
适合顺序依赖的多步流程。每步接收上步输出，逻辑清晰，易于调试。 Ideal for sequentially dependent multi-step pipelines. Each agent builds on the previous output — easy to debug.
并行扇出 · Pattern 02 Parallel Fan-out · Pattern 02
并行最快 Maximum Speed
独立子任务同时执行，N 倍提速。协调者合并所有结果，适合并行研究和多文件分析。 Independent sub-tasks run simultaneously for N× speedup. Coordinator aggregates results — great for parallel research.
层级树 · Pattern 03 Hierarchical Tree · Pattern 03
树形最强大 Most Powerful Structure
最多 5 层递归嵌套，适合复杂项目拆分。子树可独立运行，支持大规模任务分解。 Up to 5 levels of recursive nesting for complex projects. Sub-trees run independently — handles large-scale decomposition.
监督循环 · Pattern 04 Supervisor Loop · Pattern 04
循环最质量 Quality-Gated Iteration
迭代执行直到质量达标自动终止。适合代码审查、内容审编等要求高质量输出的场景。 Iterates until quality threshold is met, then exits. Ideal for code review and content editing with strict quality bars.
