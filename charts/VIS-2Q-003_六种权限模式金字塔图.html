dontAsk · orange
禁止询问用户，超出权限直接失败。No user prompts, fail if beyond scope.
用途：自动化脚本 / CIUse: automation scripts / CI
default · amber
每次敏感操作都询问。Ask for each sensitive op.
用途：日常交互式使用Use: daily interactive use
acceptEdits · emerald
自动批准文件编辑，其他仍需询问。Auto-approve edits, others ask.
用途：编辑密集型会话Use: edit-intensive sessions
auto · cyan
所有工具自动批准，不询问。All tools auto-approved.
用途：受信沙箱环境Use: trusted sandbox

绕过所有权限检查。Bypass all permission checks.
