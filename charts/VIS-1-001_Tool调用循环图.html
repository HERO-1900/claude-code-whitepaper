Tool Use 循环机制Tool Use Loop Mechanism

VIS-1-001 · 5节点环形闭环 · 6种退出条件 · 权限九步检查 VIS-1-001 · 5-node circular loop · 6 exit conditions · 9-step permission check
核心机制：Core Mechanism: Claude Code 通过 Tool Use 循环实现"能做事"的能力。AI 生成包含 tool_use block 的回复后，系统执行九步权限检查，通过后由 StreamingToolExecutor 执行工具，结果以 tool_result 追加到消息历史返回给 AI。AI 根据结果决定继续调用下一个工具或终止。满足六种退出条件之一时循环结束，输出最终 Final Response。 Claude Code gains "action capability" through the Tool Use loop. After the AI generates a response containing a tool_use block, the system runs a 9-step permission check. If approved, StreamingToolExecutor executes the tool, appends tool_result to the message history, and feeds it back to the AI. The AI then decides whether to invoke another tool or terminate. The loop ends when one of six exit conditions is met, producing the Final Response.
