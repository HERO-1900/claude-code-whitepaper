工具生命周期的四道工序（教学抽象）。一个工具从定义（buildTool 返回 plain object）到真正产出结果（handler 返回 content），概念上经过四层装配。聚合层是"工具清单"（tools.ts 里一个顶层 array），解析层负责从 assistant 消息里抽出 tool_use、用 Zod schema 校验并准备 ctx，执行层调用 handler 抓取副作用并包成统一返回格式。 Four assembly stations in the tool lifecycle (teaching abstraction). Every tool travels from definition (buildTool returning a plain object) to a produced result (handler output) through four conceptual layers. Aggregation is the "tool list" (a top-level array in tools.ts), parse extracts tool_use blocks from the assistant message and validates input via the Zod schema, and execution invokes handler and wraps side-effects into a normalised return envelope.
