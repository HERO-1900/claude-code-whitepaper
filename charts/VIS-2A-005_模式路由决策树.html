REPL — 默认路径REPL — Default Path

  • • 所有条件检查都为 No 时进入 REPL 模式REPL mode entered when all 7 checks return No

  • • 进入前会执行配置迁移检查（migrate）Migration check (migrate) runs before entering REPL

  • • SSH/Bridge/Coordinator/Direct：各对应 Feature Flag 控制SSH/Bridge/Coordinator/Direct: each controlled by feature flags
