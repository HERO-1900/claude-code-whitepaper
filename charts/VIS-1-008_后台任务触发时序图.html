主循环空闲期间自动触发的 5 类并行后台任务 5 background task types triggered automatically during main loop idle periods

投机执行 SpeculativeSpeculative Execution

主循环空闲时提前预测并预读文件。若命中则跳过 IO 等待；若未命中则丢弃（最多 1 个工具）。 Pre-reads files during idle periods. Cache hit = skip IO wait; miss = discard (max 1 tool).

上下文压缩 CompactionContext Compaction

token 超过 softLimit 时触发，阻塞主循环，将长对话压缩成摘要。压缩期间 UI 显示"正在整理记忆"。 Triggered when tokens exceed softLimit. Blocks main loop. Compresses long history into summary.

记忆整理 Memory DreamMemory Dream

会话结束时（t4 后）触发，将关键事实写入 CLAUDE.md 实现跨会话记忆持久化。 Triggered at session end (after t4). Writes key facts to CLAUDE.md for cross-session persistence.

文档维护 MagicDocsMagicDocs

自动检测文档变更并更新相关引用，保持文档与代码同步。 Auto-detects doc changes and updates references, keeping docs in sync with code.

延迟预取 PrefetchDeferred Prefetch

在后台延迟加载 OAuth 凭证、配置等低频变更资源，减少首次请求延迟。 Lazily loads OAuth tokens, configs, and other infrequently changed resources in background.
