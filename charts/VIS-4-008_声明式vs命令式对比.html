Claude Code 使用声明式 CLAUDE.md Claude Code Uses Declarative CLAUDE.md
CLAUDE.md 采用声明式设计：你只需描述期望的状态—— 项目规范、工具权限、上下文约束——而无需编写过程化脚本。 Claude 自行决定如何满足这些声明，使配置具备天然的可合并性与可预测性。 CLAUDE.md embraces declarative design: describe your desired state—project conventions, tool permissions, context constraints—without writing procedural scripts. Claude determines how to satisfy those declarations, making configuration naturally mergeable and predictable.
