核心规则 Core Rules
  • • 两个 isConcurrencySafe=true 的工具可以并行执行Two tools with isConcurrencySafe=true can execute in parallel
  • • 只要一方为 isConcurrencySafe=false，整个组合必须串行If either tool has isConcurrencySafe=false, the pair must be serial
  • • 系统信任工具自身的声明；声明错误可能导致竞态条件System trusts the tool's own declaration; wrong declaration may cause race conditions
只读工具组合 Read-only Combos
Read / Glob / Grep 互相之间均可并行，3×3 = 9 个组合全部安全，isConcurrencySafe=true Read / Glob / Grep can all run in parallel with each other — all 9 combinations (3×3) are safe, isConcurrencySafe=true
混合安全组合 Mixed-safe Combos
Agent 与只读工具可并行，Agent 本身声明了 isConcurrencySafe=true，但不属于纯只读类型 Agent can run in parallel with read-only tools — it declares isConcurrencySafe=true but is not strictly read-only
必须串行 Must Be Serial
任何涉及 Edit / Write / NBEdit / Bash 的组合都必须串行执行，这四个工具声明了 isConcurrencySafe=false Any combination involving Edit / Write / NBEdit / Bash must execute serially — these four tools declare isConcurrencySafe=false
