5个核心概念5 Core Concepts
Coordinator
主 Agent，接收用户任务并负责派发Main Agent — receives user task and dispatches sub-tasks
类比：公司 CEOAnalogy: Company CEO
Worker
子 Agent，执行 Coordinator 派发的子任务Sub-Agent — executes sub-tasks assigned by Coordinator
类比：部门经理Analogy: Department manager
Worktree
Git 独立工作目录，每个 Worker 有自己的代码副本Git working directory — each Worker has its own code copy
类比：每人一份打印资料Analogy: Individual printed copies
Scratchpad
共享内存，Agent 们读写状态用的中间区Shared memory — agents read/write shared state
类比：办公室白板Analogy: Office whiteboard
IPC
进程间通信 (Inter-Process Communication)Inter-Process Communication channel
类比：同事对讲机Analogy: Walkie-talkie
