系统架构 · Overlay 双层文件系统 Architecture · Two-Layer Overlay Filesystem
零拷贝读取Zero-copy Reads
读取未修改文件直接访问下层,无需任何复制操作。读路径开销恒为 O(1)。 Unmodified files read directly from lower — no copy overhead. Read path cost stays O(1).
安全回滚Safe Rollback
删除上层即完全回滚。如同撕掉草稿,原稿完好无损,无需逐操作 undo。 Delete upper layer = instant full rollback. Like tearing up a draft, the original remains intact.
并发隔离Concurrent Isolation
多个 Sub-Agent 各有独立上层,共享同一下层底座,互不干扰,安全并行。 Multiple sub-agents each have their own upper layer — full isolation, safe parallelism.
