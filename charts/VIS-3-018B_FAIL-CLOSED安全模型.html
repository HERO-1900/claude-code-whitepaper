安全原则：失败比危险更好 Security Principle: Safe Failure Beats Dangerous Success
Security Principle: Safe failure beats dangerous success Safe failure beats dangerous success
系统宁可无法完成任务，也不能在不确定的情况下授予权限。不确定 = 拒绝。 The system should rather fail to complete a task than grant access under uncertainty. Uncertain = Deny.
