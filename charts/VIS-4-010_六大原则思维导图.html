Claude Code 工程哲学全景 Claude Code Engineering Philosophy
Principle 1 & 4
安全优先 · 最小权限 Safety · Least Privilege
Fail-closed 默认拒绝 + 纵深防御体系。只申请必要权限，沙箱隔离工具执行，分级信任模型确保最小攻击面。 Fail-closed defaults + defense-in-depth layering. Request only necessary permissions, sandbox all tool execution, tiered trust model minimizes attack surface.
Principle 2
并发安全 Concurrency Safety
不可变状态 + 零共享可变数据。Async/Await 优先模式从根本上消除竞态条件，让并发代码可以像串行代码一样推理。 Immutable state + zero shared mutation. Async-first eliminates race conditions at the root, making concurrent code as reasoning-friendly as sequential code.
Principle 3 · 5 · 6
可组合 · 可观测 · 响应式 Compose · Observe · React
Agent = 乐高积木，Unix 哲学传承。OpenTelemetry Span + Hook 系统提供全链路可见性。事件驱动 + 流式输出保证实时响应体验。 Agents as LEGO building blocks, Unix philosophy heritage. OTel Spans + Hook system for full-stack visibility. Event-driven + streaming for real-time UX.
